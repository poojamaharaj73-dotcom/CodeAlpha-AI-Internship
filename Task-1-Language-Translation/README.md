# 🌍 Language Translation Tool

## 📌 Project Overview

The **Language Translation Tool** is a web-based application developed using Python and Streamlit that helps users translate text from one language to another.

This application uses the **Google Translator service through Deep Translator library** to provide quick and accurate translations between multiple languages.

The user can enter text, select the source language and target language, and get the translated output instantly.

---

## 🎯 Objective

- To develop an interactive language translation application.
- To understand Natural Language Processing (NLP) based translation.
- To implement language conversion using Python libraries.
- To create a simple and user-friendly web interface.

---

## ✨ Features

- 🌍 Supports multiple languages.
- 🔄 Translate text from one language to another.
- 🖥️ Interactive web interface using Streamlit.
- ⚡ Fast translation using Google Translator.
- 📝 Displays translated output instantly.
- 🚨 Provides warning when input text is empty.

---

## 🛠️ Technologies Used

- **Python**
- **Streamlit**
- **Deep Translator**
- **Google Translator**
- **Natural Language Processing (NLP)**
